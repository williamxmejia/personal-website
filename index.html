Weather Map Application

Designed a weather app that utilized JavaScript and jQuery to deliver accurate weather data to the users. This required the Mapbox API and Weather API which provided users with various weather metrics presented on Bootstrap card components along with a map and radar coverage. Initially the project contained JQuery for the GET requests using AJAX, but I decided to refactor using vanilla JavaScript to make the code more readable and modern.

Coffee Web App

Fancy Coffee Company is an online coffee store that utilizes JavaScript to display the different coffees available on the site. Fancy Coffee Company makes the experience of searching different coffees by roast and name simple, using a search bar that matches the input typed to search the site. Alongside the search feature, there is also an add feature that allows visitors to add different coffees by roast and name.The navigation bar, the coffee display and the footer were all styled using custom Bootstrap.
